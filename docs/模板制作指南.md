# 公文模板制作指南

本指南说明如何为石化公文排版工具创建和自定义 Word 模板。

## 模板系统概述

本工具使用**混合式排版**策略：
- **模板文件** (.docx) 承载固定结构：页边距、页眉页脚、命名样式定义
- **代码** 负责动态内容：正文段落生成、元数据注入

这样，格式规范改动只需修改模板文件，无需触碰代码。

## 两种创建模板的方式

### 方式一：使用内置模板生成器（推荐）

工具提供了可视化的模板生成器，可以自动生成符合标准的 Word 模板。

**操作步骤：**
1. 打开工具，点击模板选择器
2. 点击「生成 Word 模板」按钮
3. 选择标准（Q/SH 0758-2019 或 GB/T 9704-2012）
4. 选择公文类型
5. 点击「下载模板」

**支持的模板类型：**
- 红头文件（含红色分隔线、发文字号）
- 工作表单/签报
- 会议纪要
- 通知、报告、请示、批复
- 函、便笺
- 通用信函

### 方式二：手动制作模板

如需深度定制，可手动在 Microsoft Word 中创建模板。

## 手动制作模板规范

### 1. 页面设置

打开 Word → 布局 → 页面设置：

#### Q/SH 0758-2019（中国石化标准）
| 参数 | 值 |
|------|-----|
| 纸张 | A4（21cm × 29.7cm） |
| 上边距 | 3.7cm |
| 下边距 | 3.5cm |
| 左边距 | 2.8cm |
| 右边距 | 2.6cm |
| 页眉距边界 | 2.5cm |
| 页脚距边界 | 2.0cm |

#### GB/T 9704-2012（国家标准）
| 参数 | 值 |
|------|-----|
| 纸张 | A4（21cm × 29.7cm） |
| 上边距 | 2.54cm |
| 下边距 | 2.54cm |
| 左边距 | 3.17cm |
| 右边距 | 3.17cm |
| 页眉距边界 | 1.5cm |
| 页脚距边界 | 1.75cm |

### 2. 命名样式定义

**必须创建以下命名样式**（开始 → 样式 → 新建样式）：

| 样式名称 | 字体 | 字号 | 对齐 | 行距 | 首行缩进 |
|----------|------|------|------|------|----------|
| `GW_主标题` | 方正小标宋简体 | 二号（22pt） | 居中 | 固定值 28pt | 0 |
| `GW_一级标题` | 黑体 | 三号（16pt） | 左对齐 | 固定值 28pt | 2字符 |
| `GW_二级标题` | 楷体 | 三号（16pt） | 左对齐 | 固定值 28pt | 2字符 |
| `GW_三级标题` | 仿宋 | 三号（16pt） | 左对齐 | 固定值 28pt | 2字符 |
| `GW_四级标题` | 仿宋 | 三号（16pt） | 左对齐 | 固定值 28pt | 2字符 |
| `GW_正文` | 仿宋 | 三号（16pt） | 左对齐 | 固定值 28pt | 2字符 |
| `GW_主送机关` | 仿宋 | 三号（16pt） | 左对齐 | 固定值 28pt | 0 |
| `GW_落款机关` | 仿宋 | 三号（16pt） | 右对齐 | 固定值 28pt | 右缩进4字符 |
| `GW_落款日期` | 仿宋 | 三号（16pt） | 居中 | 固定值 28pt | 0 |
| `GW_附件说明` | 仿宋 | 三号（16pt） | 左对齐 | 固定值 28pt | 2字符 |
| `GW_版记` | 仿宋 | 四号（14pt） | 左对齐 | 固定值 22pt | 左右各缩进1字符 |

**注意**：样式名称必须完全一致（区分大小写）。

### 3. 占位符设置

在模板正文的对应位置插入以下占位符（纯文本）：

```
{{FILE_NUMBER}}      → 发文字号位置
{{SALUTATION}}       → 主送机关位置
{{BODY_PLACEHOLDER}} → 正文区域（必填）
{{SIGNOFF_ORG}}      → 发文机关署名位置
{{SIGNOFF_DATE}}     → 成文日期位置
{{CC_LIST}}          → 抄送机关位置（版记区域）
{{MEETING_NUMBER}}   → 纪要编号（会议纪要专用）
```

**重要提示**：
- 占位符必须单独成段，不要与其他文字混在同一行
- 不要对占位符应用任何字符格式（粗体、颜色等）
- 占位符将被完全替换，不会保留

### 4. 页眉页脚设置

#### 红头文件模板
- 插入页眉，高度 2.5cm
- 页眉内容：发文机关标志（红色，方正小标宋简体，二号）
- 页眉下方绘制红色横线（插入 → 形状 → 直线，红色 #CC0000，2.25pt）
- 插入页脚，奇偶页不同
- 页码格式：`— N —`（破折号+空格+页码+空格+破折号）
- 单数页页码右对齐，右空 1 字；双数页页码左对齐，左空 1 字

#### 其他模板
- 可省略红色分隔线
- 页脚页码设置同上

### 5. 各类型模板特殊要求

#### 红头文件
- 含发文字号占位符 `{{FILE_NUMBER}}`
- 含红色页眉横线
- 含主送机关占位符
- 含落款和日期区域
- 含版记区域

#### 工作表单/签报
- 删除红色页眉横线
- 页首添加签报表格（拟稿部门/拟稿人/电话/部门核稿/办公室核稿/签发人）
- 使用占位符：`{{DEPT}}` `{{DRAFTER}}` `{{PHONE}}` 等

#### 会议纪要
- 添加 `{{MEETING_NUMBER}}` 占位符
- 不含印章区域
- 版记区域改为「送：」/「发：」格式

## 模板注册

创建好 .docx 文件后，将其放入 `templates/` 目录，然后在 `src/constants/templateRegistry.js` 中注册：

```javascript
const TEMPLATES = {
  'my_custom_template': {
    id: 'my_custom_template',
    name: '我的自定义模板',
    description: '用于特定场景的公文格式',
    docType: 'qsh_redhead',
    standard: 'qsh_0758_2019',
    category: '自定义模板',
    fileName: '我的模板.docx',  // 对应 templates/ 目录下的文件名
    isCustom: false,
    styles: {
      title: 'GW_主标题',
      h1: 'GW_一级标题',
      h2: 'GW_二级标题',
      h3: 'GW_三级标题',
      h4: 'GW_四级标题',
      body: 'GW_正文',
      salutation: 'GW_主送机关',
      signoffOrg: 'GW_落款机关',
      signoffDate: 'GW_落款日期',
      attachment: 'GW_附件说明',
      colophon: 'GW_版记'
    }
  }
};
```

## 样式映射自定义

如果需要在不同模板中使用不同的样式名称，可以修改 `styles` 对象。例如：

```javascript
styles: {
  title: '公文标题',      // 使用自定义样式名
  h1: '一级标题',
  h2: '二级标题',
  // ... 其他样式
}
```

确保 Word 模板中存在对应的样式名称。

## 模板导入/导出

工具支持通过 JSON 格式分享自定义模板配置：

### 导出模板
1. 打开模板管理器
2. 点击「导入/导出」按钮
3. 选择要导出的模板
4. 复制 JSON 或下载文件

### 导入模板
1. 点击「导入/导出」按钮
2. 切换到「导入模板」标签
3. 粘贴 JSON 数据或上传文件
4. 点击「导入模板」

导入的模板将保存在浏览器 localStorage 中，刷新页面后仍然保留。

## 故障排查

### 模板加载失败
- 检查模板文件是否在 `templates/` 目录
- 检查 `fileName` 是否与文件名一致
- 检查模板文件是否为有效的 .docx 格式

### 样式不生效
- 检查样式名称是否与 `styles` 配置一致
- 检查样式是否在 Word 中正确定义
- 注意区分中英文标点

### 占位符不替换
- 确保占位符为纯文本，无格式
- 确保占位符单独成段
- 检查占位符拼写是否正确

### 页边距不正确
- 确认使用的是正确的标准（Q/SH 或 GB/T）
- 在 Word 中检查页面设置
- 尝试重新生成模板

## 技术细节

### Twips 单位换算
Word 使用 Twips（1/20 点）作为内部单位：
- 1 cm ≈ 567 Twips
- 1 inch = 1440 Twips
- 1 pt = 20 Twips

### XML 结构
模板本质是 ZIP 压缩的 XML 文件，包含：
- `word/document.xml` - 文档内容
- `word/styles.xml` - 样式定义
- `word/header1.xml` - 页眉
- `word/footer1.xml` - 页脚
